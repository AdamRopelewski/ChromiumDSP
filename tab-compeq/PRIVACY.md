# Privacy Policy

Tab Tone processes audio locally in the browser.

## Data Collected

Tab Tone does not collect, transmit, sell, or share personal data.

## Audio Processing

Audio from the active tab is captured only after the user starts capture. Audio is routed through Web Audio inside Chrome and is not sent to any server.

## Local Storage

DSP settings such as EQ, compressor, limiter, gain, and width are stored in Chrome local extension storage. These settings stay on the user's device.

## Network Requests

Tab Tone does not make external network requests.

## Contact

For privacy questions, open an issue in the project repository.
